# Central de Vendas — Dashboard em Tempo Real

Dashboard de vendas em tempo real para agência de mídia/publicidade, com atualizações instantâneas via WebSocket e integração com backend Node.js. O frontend exibe KPIs, gráfico por hora, ranking de clientes, feed de vendas em tempo real e status de conexão.

## Como executar

1. Instalar dependências:

```bash
npm install
```

2. Iniciar o servidor:

```bash
node server.js
```

3. Acessar no navegador:

```
http://localhost:3000
```
